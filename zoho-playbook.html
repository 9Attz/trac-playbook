RB — Retail Business Retail Business

📋 Reports ที่ใช้บ่อย

Report Nameใช้ทำอะไรความถี่
Daily Fleet Inventory Reportดูสถานะรถทั้งหมดว่า Available / Rented / Under Repair ประจำวันทุกวัน
Fleet Utilization EHIวัด Utilization % แยกตาม Car Group, Branchรายสัปดาห์
Agreement Check Outรายการ Check-out ทั้งหมด ดูลูกค้าที่รับรถวันนี้ทุกวัน
A Collectionติดตามยอดเก็บเงิน ลูกค้าค้างชำระรายสัปดาห์
A Aging By Due Dateวิเคราะห์ Aging หนี้ตามวันครบกำหนดรายสัปดาห์
Damage Analysisสรุปความเสียหายรถแยกประเภท, Branchรายเดือน
A Invoice Receiptใบเสร็จรับเงิน ตรวจสอบการชำระทุกวัน
Sale Code Rental (This Month)วิเคราะห์ยอดขายแยกตาม Sale Code เดือนนี้รายเดือน

CB — Corporate Business Corporate Business

📋 Reports ที่ใช้บ่อย

Report Nameใช้ทำอะไรความถี่
Lease Agreement Reportดูรายการสัญญาเช่าระยะยาวทั้งหมด สถานะ, วันหมดสัญญาทุกวัน
Lease Invoice Listรายการ Invoice Leasing ที่ออกแล้ว ยังไม่ได้ชำระรายสัปดาห์
A HP Loan Detailรายละเอียดสินเชื่อ HP แยกตามสัญญา, งวดที่ค้างชำระรายสัปดาห์
A HP Loan Totalยอดรวม HP Loan แยกตาม Portfolioรายเดือน
Lease Projected Revenueประมาณรายได้ในอนาคตจากสัญญา Lease ที่มีอยู่รายเดือน
Lease Quotationรายการใบเสนอราคา Lease ที่รอการตัดสินใจรายสัปดาห์
A Sale Code Leasingยอดขาย Leasing แยกตาม Sale Codeรายเดือน
Debitor Credit Limitsวงเงินลูกค้า Corporate ที่ได้รับอนุมัติ vs ที่ใช้ไปแล้วรายสัปดาห์
A Sale TAX (Yearly)ภาษีขายรายปี สำหรับทำรายงานบัญชีรายเดือน/ปี

Formula Column Analyst

เปิด Table → Add → Formula Column → ตั้งชื่อ → เขียน Expression

FormulaอธิบายBU
("rented"/"total_fleet")*100Utilization %RB
datediff("return_date","checkout_date")จำนวนวันเช่าRB
if("days_overdue"<=30,'0-30',if("days_overdue"<=60,'31-60','60+'))Aging bucketCB

Query Table (SQL) Analyst

Data → "+ New Table" → "Query Table"
เขียน SQL → กด Run Preview → ตั้งชื่อ → Save
-- RB: Utilization รายสาขา เดือนนี้ SELECT b.branch_name, COUNT(DISTINCT f.car_id) AS total_fleet, COUNT(DISTINCT a.car_id) AS rented_cars, ROUND(COUNT(DISTINCT a.car_id) * 100.0 / NULLIF(COUNT(DISTINCT f.car_id),0), 1) AS utilization_pct FROM Fleet_Inventory f LEFT JOIN Agreement_Rental a ON f.car_id = a.car_id AND MONTH(a.checkout_date) = MONTH(NOW()) JOIN Data_Branch b ON f.branch_code = b.branch_code GROUP BY b.branch_name ORDER BY utilization_pct DESC
⚠️
Query Table เป็น Virtual Table — Re-execute ทุกครั้งที่เปิด Report ควรมี WHERE clause เสมอเพื่อลด load
🖥️

FAQ ทุก Role
