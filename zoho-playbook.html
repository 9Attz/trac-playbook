ภาพรวมระบบ ทุก Role

Zoho Analytics คือแพลตฟอร์ม Business Intelligence (BI) ที่ใช้สำหรับวิเคราะห์ข้อมูล สร้าง Report และ Dashboard แบบ Interactive สำหรับองค์กร TRAC ได้ติดตั้ง Workspace ชื่อ TRACSaaS_PROD ซึ่งเชื่อมต่อกับฐานข้อมูลหลักของระบบ

📊

รายงานแบบ Real-time

ข้อมูลอัปเดตอัตโนมัติจากฐานข้อมูล ไม่ต้องรอรายงาน manual

🖥️

Dashboard แบบ Interactive

คลิกกรองข้อมูล เจาะลึก (Drill-down) ได้โดยตรงบนหน้าจอ

🤖

AI — Ask Zia

ถามคำถามเป็นภาษาธรรมชาติ ระบบจะสร้างกราฟให้อัตโนมัติ

🔒

Row-level Security

ควบคุมข้อมูลที่แต่ละ User มองเห็นได้ในระดับ Row

📥

Export หลายรูปแบบ

รองรับ Excel, PDF, CSV, Image และ Email Scheduling

🔌

เชื่อมต่อหลาย Data Source

รองรับ MySQL, MS SQL, Google Sheets, REST API และอื่นๆ

เข้าสู่ระบบ User

เปิดเว็บเบราว์เซอร์ ไปที่ analytics.zoho.com หรือลิงก์ที่ Admin แชร์ให้
คลิก "Sign In" → ใส่อีเมลองค์กร (Zoho Account) แล้วกด Next
ใส่ Password → คลิก Sign In (หากมี 2FA ให้กรอก OTP จากแอปหรืออีเมล)
หลังเข้าสู่ระบบ จะเห็นหน้า Home → เลือก Workspace TRACSaaS_PROD
⚠️
หากยังไม่มีบัญชี ให้ติดต่อ Admin เพื่อขอสิทธิ์ ไม่ควรสมัครบัญชีใหม่เองโดยไม่ได้รับอนุญาต
📊

ดู Dashboard User

ใน Workspace คลิกเมนู Dashboards ที่แถบซ้าย
เลือก Dashboard ที่ต้องการจากรายการ — สังเกต icon 📊 สีน้ำเงิน
หน้า Dashboard จะแสดง Widget หลายชิ้น เช่น กราฟ, ตัวเลข KPI, ตาราง
คลิกที่กราฟหรือตัวเลขใดก็ได้ เพื่อ Drill-down ดูรายละเอียดเพิ่มเติม
ใช้ Date Filter หรือ Dropdown Filter ที่มุมบนขวาของ Dashboard เพื่อเปลี่ยนช่วงเวลา
💡
กด F11 หรือ Full Screen button เพื่อแสดง Dashboard แบบเต็มจอ เหมาะสำหรับ TV Dashboard ในออฟฟิศ
📋

ดู Report User

คลิกเมนู Reports ที่แถบซ้าย หรือเข้าผ่าน Explorer แล้วกรอง Type = Report
เลือก Report ที่ต้องการ — Report อาจเป็น Chart, Table หรือ Pivot ก็ได้
หากเป็น Table Report สามารถ คลิกหัวคอลัมน์ เพื่อเรียงลำดับข้อมูล
กด Apply Filter (icon รูปกรวย) เพื่อกรองข้อมูลตามเงื่อนไขที่ต้องการ
🔍

กรองข้อมูล (Filter) User

ประเภท Filterวิธีใช้ตัวอย่าง
User Filterกด icon กรวย → เลือกค่ากรองตาม Branch, ประเภทรถ
Date Rangeคลิก Date Picker ที่ Widgetเดือนที่ผ่านมา, ปีนี้
Quick Filterคลิกที่ Legend หรือ Bar บนกราฟคลิกเลือก Region ใน Pie Chart
URL Filterเพิ่ม parameter ท้าย URL (สำหรับ Embed)?ZOHO_CRITERIA=branch="BKK"
📥

Export ข้อมูล User

เปิด Report หรือ Dashboard ที่ต้องการ Export
คลิกไอคอน ⋮ (More Options) หรือ Export ที่มุมขวาบน
เลือกรูปแบบ: Excel (.xlsx) / PDF / CSV / Image (.png)
เลือกตัวเลือกเพิ่มเติม เช่น Export ทุกหน้า หรือเฉพาะหน้าที่เห็น แล้วกด Export
✅
สำหรับ Report ที่ต้องส่งอัตโนมัติทุกสัปดาห์ ให้ขอให้ Analyst หรือ Admin ตั้งค่า Email Scheduling ให้

Ask Zia — AI Assistant User

Ask Zia คือ AI ของ Zoho ที่รับคำถามภาษาธรรมชาติ (ภาษาอังกฤษ) แล้วสร้าง Chart หรือตอบคำถามจากข้อมูลใน Workspace โดยอัตโนมัติ

คลิกเมนู Ask Zia ที่แถบซ้าย
เลือก Table ที่ต้องการถามจาก Dropdown
พิมพ์คำถามเป็นภาษาอังกฤษ เช่น "Show me total revenue by branch this month"
Zia จะสร้างกราฟให้อัตโนมัติ สามารถกด "Add to Dashboard" เพื่อบันทึกได้
ตัวอย่างคำถาม Ziaความหมาย
Top 10 customers by revenueลูกค้า 10 อันดับแรกตามรายได้
Monthly agreement count last yearจำนวนสัญญาแต่ละเดือนปีที่แล้ว
Fleet utilization by car groupอัตราใช้งานรถแยกตามกลุ่มรถ
Overdue invoices count by branchจำนวน Invoice เกินกำหนดแยก Branch

Permission & Row-level Security Admin

Zoho Analytics รองรับการควบคุมข้อมูลระดับ Row ทำให้ User แต่ละคนเห็นข้อมูลเฉพาะส่วนที่เกี่ยวข้องกับตัวเอง

ไปที่ Data → เลือก Table → คลิก ⋮ → Share
เลือก User หรือ Group → กำหนดระดับ: Read / Read & Write
เพื่อตั้ง Row-level Security: คลิก "Row-level Security" → "Add Criteria"
ตั้งเงื่อนไข เช่น branch_code = [USER_EMAIL] → บันทึก
💡
ใช้ User Attribute เช่น ZOHO_USER_EMAIL ในเงื่อนไข Row-level Security เพื่อให้ระบบ filter ข้อมูลโดยอัตโนมัติตามผู้ Login
⚙️

Formula Column Analyst

Zoho Analytics รองรับการสร้าง Calculated Column ใน Table เพื่อคำนวณค่าใหม่โดยไม่ต้องแก้ข้อมูลต้นทาง

เปิด Table → คลิก "Add → Formula Column"
พิมพ์ชื่อคอลัมน์ → เขียน Formula ในช่อง Expression
คลิก Save — คอลัมน์ใหม่จะแสดงในทุก Report ที่ใช้ Table นี้
ตัวอย่าง Formulaอธิบาย
"Revenue" - "Cost"คำนวณ Gross Profit
"Revenue" / "Days" * 30ประมาณรายได้ต่อเดือน
datediff("return_date", "checkout_date")จำนวนวันที่เช่า
if("status" = 'Active', 1, 0)Flag ว่า Active หรือไม่
concat("branch_code", " - ", "branch_name")รวมสอง Column เป็น Label
year("agreement_date")ดึงปีจาก Date Column
-- ตัวอย่าง: คำนวณ Utilization % ของรถ if("total_fleet" > 0, ("rented_cars" / "total_fleet") * 100, 0 ) -- ตัวอย่าง: จัดกลุ่ม Aging if("days_overdue" <= 30, '0-30 วัน', if("days_overdue" <= 60, '31-60 วัน', if("days_overdue" <= 90, '61-90 วัน', '90+ วัน')))

Query Table (SQL) Analyst

Query Table ช่วยให้เขียน SQL เพื่อ Join, Filter หรือ Transform ข้อมูลจากหลาย Table พร้อมกัน ผลลัพธ์จะเป็น Virtual Table ที่ใช้สร้าง Report ได้

ไปที่ Data → "+ New Table" → "Query Table"
เขียน SQL ในช่อง Query Editor
คลิก Run เพื่อ Preview ผลลัพธ์ 50 แถวแรก
ตั้งชื่อ → คลิก Save → ใช้เป็น Table ปกติในการสร้าง Report
-- ตัวอย่าง: Report สรุปยอดเช่าแต่ละ Branch เดือนนี้ SELECT b.branch_name, COUNT(a.agreement_id) AS total_agreements, SUM(a.total_amount) AS total_revenue, AVG(a.rental_days) AS avg_rental_days FROM Agreement_Rental a JOIN Data_Branch b ON a.branch_code = b.branch_code WHERE MONTH(a.agreement_date) = MONTH(NOW()) AND YEAR(a.agreement_date) = YEAR(NOW()) GROUP BY b.branch_name ORDER BY total_revenue DESC
⚠️
Query Table เป็น Read-only Virtual Table — ไม่สามารถ Edit ข้อมูลโดยตรงได้ และจะ Re-execute SQL ทุกครั้งที่ Report ถูกเปิด
🖥️

FAQ — คำถามที่พบบ่อย ทุก Role

คำถามคำตอบ
ข้อมูลใน Report ไม่อัปเดต?ตรวจสอบ Last Sync Time ที่ Data Source → กด Sync Now เพื่อ Force update
ไม่เห็นบาง Table หรือ Report?ติดต่อ Admin เพื่อขอสิทธิ์ — อาจเป็นเรื่อง Permission ที่ยังไม่ได้กำหนด
Export Excel แล้วข้อมูลหาย?Zoho Export ได้สูงสุด 1 ล้านแถว ถ้าเกินให้ใช้ CSV หรือขอ Admin Export ผ่าน API
Formula คำนวณผิด?ตรวจสอบประเภทข้อมูล (Data Type) ของคอลัมน์ — ตัวเลขที่เก็บเป็น Text จะคำนวณไม่ได้
Dashboard โหลดช้า?ลดจำนวน Widget, ตั้ง Filter ที่เหมาะสม หรือใช้ Query Table แทนการ Join ใน Report
จะ Embed Dashboard ในเว็บได้ไหม?ได้ — Share → Publish → Get Embed Code แล้วนำ <iframe> ไปวางในเว็บ

Tips & Tricks ทุก Role

⌨️

Keyboard Shortcuts

Ctrl+S บันทึก Report / Dashboard
Ctrl+Z Undo การแก้ไข
F5 Refresh ข้อมูลใน Report

📌

Pin รายการโปรด

คลิกขวาที่ Report หรือ Dashboard → Add to Favorites เพื่อเข้าถึงได้รวดเร็วจาก Home

📧

Email Scheduling

เปิด Report → Share → Schedule → ตั้งเวลาส่งอีเมลอัตโนมัติ รองรับ Daily / Weekly / Monthly

📱

Zoho Analytics App

ดาวน์โหลด App บน iOS / Android เพื่อดู Dashboard บนมือถือได้ทุกที่ทุกเวลา

🎨

Custom Theme

Dashboard → Edit → Theme เลือกสีหรือ Upload Logo เพื่อให้ตรง Branding องค์กร

🔗

Drill-down Link

คลิก Bar ในกราฟ → Configure Drill Down → ลิงก์ไปยัง Report หรือ Dashboard อื่นได้
