TRAC IT  /  Training Roadmap

0 → Pro ใน
8 สัปดาห์

แผนพัฒนาทักษะ Claude AI สำหรับ IT Admin และ IT Support — 4 phases, 20+ exercises, daily habits

Duration 8 สัปดาห์ Level มือใหม่ → Pro By Attapon Pinpleumjit / TRAC IT

Learning Path

4 Phases สู่ IT Pro

แต่ละ phase ใช้เวลา 2 สัปดาห์ ทำได้วันละ 20-30 นาที เรียนรู้จากงานจริงที่ทำอยู่ทุกวัน

Week 1–2

PHASE 1 — Foundation

เริ่มต้น & ทดลอง

เรียนรู้พื้นฐาน Claude ทดลองกับงาน IT ง่ายๆ ที่ทำอยู่ทุกวัน

01

สมัคร Claude.ai และทดลองถามปัญหา IT ง่ายๆ 5 ข้อที่เจอในสัปดาห์นี้จริงๆ

02

สร้าง Project "TRAC IT" ใส่ Custom Instructions: OS, network environment, tools ที่ใช้ใน TRAC

03

Troubleshoot แรก เอา error message จริงที่เจอวันนี้ paste ให้ Claude วิเคราะห์ วัดเวลาที่ประหยัดได้

04

อัพโหลด Log File นำ log จริง (anonymized) อัพโหลด ให้ Claude สรุปปัญหาที่พบ

05

เขียน KB Article แรก ใช้ Claude เขียน Knowledge Base article สำหรับปัญหาที่ตอบบ่อยที่สุด

Week 3–4

PHASE 2 — Skill Building

Documentation & Scripting

ฝึกเขียนเอกสาร IT และ script พื้นฐาน สร้าง Prompt Library ของตัวเอง

06

เขียน Runbook แรก ใช้ Claude ร่าง Runbook กระบวนการ IT 1 อย่างที่ทำบ่อยที่สุด

07

สร้าง PowerShell Script ขอให้ Claude เขียน script ตรวจ disk space และส่ง email alert

08

Test Script บน Lab รัน script ที่ Claude เขียนบน test environment บันทึกผลและปรับแก้

09

ตอบ Helpdesk Ticket ใช้ Claude draft คำตอบ ticket 10 ใบในสัปดาห์นี้ วัดเวลาก่อน-หลัง

10

สร้าง IT Prompt Library รวม prompt IT ที่ใช้งานได้จริงอย่างน้อย 10 prompts

Week 5–6

PHASE 3 — Advanced

Security, Policy & Dashboard

ก้าวสู่งาน IT ขั้นสูง: security policy, log analysis, และ IT dashboard

11

ร่าง IT Security Policy ใช้ Claude ร่าง Password Policy หรือ Access Control Policy สำหรับ TRAC

12

วิเคราะห์ Log อัตโนมัติ อัพโหลด Event Viewer log (anonymized) ให้ Claude หา top 5 issues

13

สร้าง IT Dashboard ใช้ Cowork สร้าง dashboard แสดง ticket volume, MTTR, system uptime

14

เปรียบเทียบ Vendor ใช้ Claude เปรียบเทียบ 2 solutions ที่กำลังพิจารณาอยู่ สร้าง evaluation matrix

15

เขียน Script ซับซ้อน สร้าง automation script ที่ทำงานหลายขั้นตอน เช่น user provisioning

Week 7–8

PHASE 4 — Pro Level

Automation & Knowledge Sharing

สร้าง IT workflow อัตโนมัติและแชร์ความรู้ให้ทีม TRAC IT

16

สร้าง IT SOP ครบชุด เขียน SOP 3 กระบวนการที่สำคัญที่สุดใน TRAC IT ด้วย Claude

17

สร้าง IT Onboarding Kit ออกแบบ onboarding checklist และ guide สำหรับพนักงานใหม่ที่ต้องใช้ IT

18

สร้าง IT Monthly Report ใช้ Claude สร้าง IT operations report สำหรับผู้บริหาร

19

Teach the Team สอนเพื่อนร่วมทีม IT 1 คน ใช้เวลา 30 นาที ให้เขาลอง troubleshoot ด้วย Claude

20

Measure & Document Impact รวบรวมตัวเลขเวลาที่ประหยัดได้ใน 8 สัปดาห์ สร้าง ROI report

Daily Habits

นิสัย IT Pro ที่ต้องสร้าง

ทำสิ่งเหล่านี้ทุกวัน สร้าง muscle memory ในการใช้ Claude กับงาน IT

🖥️

เช้า — Check Alerts (5 นาที)

นำ alert/error ที่ค้างอยู่มาถาม Claude ก่อนเริ่มวัน วางแผนการ troubleshoot

⚡

ระหว่างวัน — Paste First (ทันที)

เมื่อเจอ error ใดๆ paste ให้ Claude ก่อนเสมอ แม้จะรู้วิธีแก้แล้ว เพื่อฝึกนิสัย

💾

หลังแก้ปัญหา — Document It (3 นาที)

ให้ Claude เขียน KB article ทุกครั้งที่แก้ปัญหาสำเร็จ สะสม knowledge base

📊

ทุกศุกร์ — Time Tracking (10 นาที)

บันทึกว่าสัปดาห์นี้ใช้ Claude แก้ปัญหาอะไร ประหยัดเวลาได้กี่ชั่วโมง

Week 1 Checklist

เริ่มต้น 7 วันแรก

ทำให้ครบในสัปดาห์แรก สร้าง foundation ที่แข็งแกร่งสำหรับ 7 สัปดาห์ต่อไป

วันที่ 1: สมัคร claude.ai เอา error message จริงจากวันนี้ถาม Claude ดูผลลัพธ์

วันที่ 2: สร้าง Project "TRAC IT" ใส่ environment ของคุณ: OS, network, tools

วันที่ 3: อัพโหลด log file จริง (anonymized) ให้ Claude วิเคราะห์

วันที่ 4: ขอให้ Claude เขียน PowerShell/Bash script งานเล็กๆ 1 อย่าง

วันที่ 5: เขียน KB article ด้วย Claude สำหรับปัญหาที่ตอบบ่อยที่สุด

วันที่ 6: บันทึก prompt ที่ได้ผลดี 3 prompts ลง Prompt Library

วันที่ 7: แชร์ให้เพื่อนทีม IT 1 คน ลองใช้ Claude troubleshoot ด้วยกัน

Ready?

เริ่ม Phase 1 วันนี้เลย

เปิด claude.ai → เอา error ที่เจอวันนี้ paste เลย ใช้เวลาแค่ 5 นาที
